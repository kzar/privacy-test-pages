Simple test page that serves as a quick visual test to show if the Request Blocklist feature is active and working.

Test cases are shown as green if they are blocked/allowed as expected, and shown as red otherwise. Where possible, "Loaded"/"Blocked" text is also shown.

Notes:

  • Some browsers will display blocked frames as white/grey, instead of the green/red background that we want. Consider that a pass for the "Blocked ..." test cases, and a fail for the others.
  • For automated integration tests, prefer the Request Blocking Test Page. An empty blocklist + suitable Request Blocklist configuration can be used to test the feature.
